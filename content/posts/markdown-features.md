# Markdown Features

Here are some of the Markdown features supported by this blog:

## Lists

Unordered list:
- Item one
- Item two
- Item three

Ordered list:
1. First item
2. Second item
3. Third item

## Code Blocks

Inline `code` and code blocks:

```python
def hello():
    print("Hello from Python!")
```

```javascript
function hello() {
    console.log("Hello from JavaScript!");
}
```
